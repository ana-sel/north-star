1. Intake Filter Before Active Cards

Raw thoughts are saved first. Only filtered items become active cards. Fake wants are archived as insights, not lost.

Gate before planning
1. Raw Thought

“I want a Ferrari.” “Go to Japan with family.” “Pay bill.”

2. Want / Need

Want, need, obligation, random impulse, someone else’s expectation.

3. Mission Filter

Happiness, freedom, clarity, self-refinement, solitude, meaning, contribution.

4. Exit Decision

Keep, later, archive insight, delete, delegate, split, clarify.

Important design rule

The filter is not a punishment. It is a truth gate. Even rejected wants become useful self-knowledge if archived properly.

Protects freedom and budget.

Daily, weekly, monthly feedback loops.

Agent council for “What should I do today?”

Mission checks alignment. Time checks capacity. Energy checks intensity. Money checks urgency. Productivity checks bottlenecks. Focus chooses the final 1–3 actions.

Backend location

Privacy rule
